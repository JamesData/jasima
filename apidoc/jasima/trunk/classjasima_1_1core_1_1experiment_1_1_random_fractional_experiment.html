Detailed Description

Allows to a certain number of experiments chosen randomly. This number of experiments run is usually smaller than the number of possible factor combinations, i.e. only a fraction of possible designs is tested.

Torsten Hildebrandt hil@b.nosp@m.iba..nosp@m.uni-b.nosp@m.reme.nosp@m.n.de, 2012-06-08
