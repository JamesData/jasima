Detailed Description

Base class implementing the Abstract Factory pattern. Using classes always call ExecutorFactory.getExecutor(). This call is delegated to a non-abstract implementation of ExecutorFactory. Which ExecutorFactory to use is determined by a system property "jasima.core.ExecutorFactory", as a default ThreadPoolExecutor with a number of threads equal to the number of available processors is used.
