Detailed Description

Base class for shop experiments. This class wraps a SimpleJobShop. Derived classes will typically populate the shop with machines and JobSources and add functionality to collect some statistics and produce appropriate experiment results.

Author
Torsten Hildebrandt, 2010-03-12
